# Ruby完全ガイド

Rubyの実践的な実装方法を、実務で使える実装例とベストプラクティスとともに詳しく解説します。

## 1. Rubyとは

### Rubyの特徴

Rubyは、読みやすく、書きやすいプログラミング言語です。

```
Rubyの特徴
   ├─ 読みやすさ
   ├─ 生産性
   ├─ 柔軟性
   └─ オブジェクト指向
```

## 2. 基本構文

### 変数と型

```ruby
# 変数の定義
name = "Alice"
age = 30
height = 165.5
is_active = true

# 型の確認
puts name.class  # String
puts age.class   # Integer
puts height.class  # Float
puts is_active.class  # TrueClass
```

### データ構造

```ruby
# 配列
fruits = ["apple", "banana", "orange"]
fruits << "grape"
puts fruits[0]  # apple

# ハッシュ
user = {
  name: "Alice",
  age: 30,
  email: "alice@example.com"
}
puts user[:name]  # Alice

# シンボル
status = :active
```

### メソッド

```ruby
def greet(name, age = 0)
  if age > 0
    "Hello, #{name}! You are #{age} years old."
  else
    "Hello, #{name}!"
  end
end

result = greet("Alice", 30)
puts result  # Hello, Alice! You are 30 years old.
```

### クラス

```ruby
class User
  attr_accessor :name, :email
  
  def initialize(name, email)
    @name = name
    @email = email
  end
  
  def info
    "#{@name} (#{@email})"
  end
end

user = User.new("Alice", "alice@example.com")
puts user.info  # Alice (alice@example.com)
```

## 3. ブロックとイテレータ

```ruby
# ブロック
[1, 2, 3, 4, 5].each do |num|
  puts num * 2
end

# map
squared = [1, 2, 3, 4, 5].map { |x| x ** 2 }
puts squared  # [1, 4, 9, 16, 25]

# select
even_numbers = [1, 2, 3, 4, 5].select { |x| x.even? }
puts even_numbers  # [2, 4]
```

## 4. モジュール

```ruby
module Loggable
  def log(message)
    puts "[#{Time.now}] #{message}"
  end
end

class User
  include Loggable
  
  def create
    log("Creating user")
    # ユーザー作成の処理
  end
end
```

## 5. エラーハンドリング

```ruby
begin
  result = 10 / 0
rescue ZeroDivisionError => e
  puts "Error: #{e.message}"
rescue => e
  puts "Unexpected error: #{e.message}"
else
  puts "No errors occurred"
ensure
  puts "This always executes"
end
```

## 6. 実践的なベストプラクティス

### 命名規則

```ruby
# 良い例
class UserService
  def create_user(user_data)
    # 処理
  end
end

# 悪い例
class us
  def cu(d)
    # 処理
  end
end
```

### ドキュメント

```ruby
# メソッドのドキュメント
# @param name [String] ユーザー名
# @param age [Integer] 年齢
# @return [String] 挨拶メッセージ
def greet(name, age)
  "Hello, #{name}! You are #{age} years old."
end
```

## まとめ

Ruby完全ガイドのポイント：

- **基本構文**: 変数、型、データ構造、メソッド、クラス
- **ブロックとイテレータ**: each、map、select
- **モジュール**: コードの再利用
- **エラーハンドリング**: begin-rescue-ensure
- **ベストプラクティス**: 命名規則、ドキュメント

適切なRubyの使用により、効率的な開発が可能になります。

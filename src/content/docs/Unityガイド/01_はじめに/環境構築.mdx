# Unity環境構築ガイド 🎮

Unityでのゲーム開発を始めるための環境構築手順を、初心者にも分かりやすく解説します。

## 1. Unity Hubのインストールとアカウント設定

- **Unity Hubのダウンロードとインストール**  
  Unity公式サイトからUnity Hubをダウンロードし、インストールします。

- **アカウントの作成とサインイン**  
  Unity Hubを起動したら、Unityアカウントを作成し、サインインします。

- **ライセンスのアクティベート**  
  ログイン後、ライセンスをアクティベートします。個人利用であれば無料で取得できる「Unity Personal」を選択してください。

## 2. Unityエディターと追加モジュールのインストール

- **Unityエディターのインストール**  
  Unityアカウントの設定が完了したら、開発に使うUnityエディターをインストールします。Unity Hubの左メニューから「Installs」を選択し、「Install Editor」ボタンをクリックします。長期的な安定性が保証されているLTS (Long Term Support)版のインストールを推奨します。

- **追加モジュールの選択**  
  インストールする際に、以下のモジュールを追加で選択します。
  - Microsoft Visual Studio Community: スクリプト開発に必須のIDEです。
  - Android Build Support: Androidアプリ向けにビルドする場合に必要です。
  - iOS Build Support: iOSアプリ向けにビルドする場合に必要です。

## 3. Visual Studioのセットアップと連携

- **Visual Studioの設定確認**  
  Visual Studioのインストーラーを開き、「ゲーム開発」ワークロードがインストールされていることを確認します。

- **Unityとの連携設定**  
  Unityエディターを起動し、「Edit」→「Preferences」 (macOSでは「Unity」→「Settings」) を開きます。「External Tools」で「External Script Editor」に「Visual Studio」が設定されていることを確認します。これで、Unity上でスクリプトファイルをダブルクリックすると自動でVisual Studioが起動するようになります。

## プロジェクト作成と初期設定

- **新しいプロジェクトの作成**  
  Unity Hubの左メニューから「Projects」を選択し、「New Project」ボタンをクリックします。開発したいゲームに合わせてテンプレートを選択します。
  - 2D、3D: 基本的なゲーム開発用。
  - URP (Universal Render Pipeline): パフォーマンスを重視したグラフィックス用。
  - HDRP (High Definition Render Pipeline): 高品質なグラフィックス用。

- **プロジェクトの初期設定**  
  エディターが立ち上がったら、すぐに開発を始められるように初期設定を行います。
  - レイアウトの調整: 画面右上のドロップダウンから「2 by 3」や「Tall」などのレイアウトを試して、作業しやすい配置を見つけましょう。
  - ゲームオブジェクトの作成: 「Hierarchy」ウィンドウ内で右クリックし、3D Object > Cubeなどを選択すると、シーンにオブジェクトを配置できます。
  - スクリプトの作成: 「Project」ウィンドウ内で右クリックし、Create > C# Scriptを選択すると、新しいスクリプトファイルを作成できます。

## 開発をサポートする追加情報

- **推奨されるPC環境**  
  Unityを快適に動作させるには、ある程度のPCスペックが必要です。
  - OS: Windows 10 (64-bit) / macOS Mojave 10.14+
  - CPU: Intel Core i7 / AMD Ryzen 7 もしくはそれ以上
  - RAM: 16GB もしくはそれ以上
  - ストレージ: SSD (256GB以上)

- **アセットストアの活用**  
  Unityのアセットストアには、3Dモデル、テクスチャ、スクリプトなど、開発を助ける様々な素材が公開されています。これらを活用することで、開発の幅が大きく広がります。

- **パッケージの追加**  
  「Window」→「Package Manager」から、Unityが提供する追加機能（ポストプロセッシング、UI機能など）をインストールできます。

## セキュリティ向け拡張機能

セキュリティテストと脆弱性検出に役立つおすすめのChrome拡張機能を紹介します。

### 1. Security Headers

HTTPレスポンスヘッダーのセキュリティ設定を確認する拡張機能です。

#### 機能

- **セキュリティヘッダーの確認**: CSP、HSTS、X-Frame-Optionsなどを確認
- **セキュリティスコア**: セキュリティヘッダーのスコアを表示
- **推奨事項の表示**: セキュリティヘッダーの推奨事項を表示

#### 使い方

1. Chrome Web Storeから「Security Headers」をインストール
2. アイコンをクリックしてセキュリティヘッダーを確認

### 2. Cookie-Editor

Cookieを管理・編集する拡張機能です。

#### 機能

- **Cookieの表示**: 現在のページのCookieを表示
- **Cookieの編集**: Cookieの値を編集
- **Cookieの削除**: Cookieを削除
- **Cookieのエクスポート**: CookieをJSON形式でエクスポート

#### 使い方

1. Chrome Web Storeから「Cookie-Editor」をインストール
2. アイコンをクリックしてCookieを管理

### 3. Application Inspector

Webアプリケーションのストレージ情報を確認する拡張機能です。

#### 機能

- **LocalStorageの確認**: LocalStorageの内容を確認
- **SessionStorageの確認**: SessionStorageの内容を確認
- **IndexedDBの確認**: IndexedDBの内容を確認
- **Cache Storageの確認**: Cache Storageの内容を確認

#### 使い方

1. Chrome Web Storeから「Application Inspector」をインストール
2. アイコンをクリックしてストレージ情報を確認

### まとめ

セキュリティ向けのおすすめ拡張機能：

1. **Security Headers**: セキュリティヘッダーの確認
2. **Cookie-Editor**: Cookieの管理・編集
3. **Application Inspector**: ストレージ情報の確認

これらの拡張機能を活用することで、セキュリティテストと脆弱性検出を効率化できます。

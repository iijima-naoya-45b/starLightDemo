Next.jsでのキャッシュ戦略を以下に示します。

## 静的ファイルのキャッシュ

Next.jsは、静的ファイルを自動的にキャッシュします。これにより、ページの読み込み速度が向上します。

## ISR（Incremental Static Regeneration）

ISRを使用して、静的ページを定期的に再生成し、最新のデータを提供します。

```javascript
export async function getStaticProps() {
  return {
    props: {},
    revalidate: 60, // 60秒ごとに再生成
  };
}
```

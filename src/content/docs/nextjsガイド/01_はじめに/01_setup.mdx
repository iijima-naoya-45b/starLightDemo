---
title: "Next.jsのセットアップ"
label: "Next.jsのセットアップ"
---

Next.jsを使用するには、Node.jsが必要です。以下の手順でNext.jsプロジェクトをセットアップします。

```bash
# 新しいNext.jsプロジェクトの作成
npx create-next-app@latest my-next-app

# プロジェクトディレクトリに移動
cd my-next-app

# 開発サーバーの起動
npm run dev
```

これで、Next.jsの開発環境が整いました。`http://localhost:3000`でアプリケーションを確認できます。

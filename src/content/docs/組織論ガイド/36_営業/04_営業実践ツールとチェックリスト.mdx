### 3.2 ROI計算テンプレート

#### 投資対効果の定量化

```markdown
## ROI計算テンプレート

### 💰 投資額（初期費用）
| 項目 | 金額 | 備考 |
|------|------|------|
| システム導入費 | ¥5,000,000 | ライセンス・設定費 |
| ハードウェア | ¥2,000,000 | サーバー・ネットワーク |
| 導入支援費 | ¥1,500,000 | コンサル・研修費 |
| **合計初期投資** | **¥8,500,000** | |

### 📈 年間効果（コスト削減・売上向上）
| 効果項目 | 年間効果額 | 計算根拠 |
|----------|------------|----------|
| 業務効率化 | ¥3,000,000 | 作業時間50%削減 |
| 人件費削減 | ¥2,000,000 | 1名分の工数削減 |
| エラー削減 | ¥1,000,000 | 品質向上効果 |
| 売上向上 | ¥2,000,000 | 新規顧客獲得 |
| **年間効果合計** | **¥8,000,000** | |

### 📊 ROI分析
- **年間ROI** = (年間効果 - 年間運用費) ÷ 初期投資 × 100
- **投資回収期間** = 初期投資 ÷ (年間効果 - 年間運用費)
- **3年間累積効果** = (年間効果 - 年間運用費) × 3年 - 初期投資

### 計算例
- 年間運用費：¥1,000,000
- 年間純効果：¥8,000,000 - ¥1,000,000 = ¥7,000,000
- 年間ROI：¥7,000,000 ÷ ¥8,500,000 × 100 = 82.4%
- 投資回収期間：¥8,500,000 ÷ ¥7,000,000 = 1.2年
- 3年間累積効果：¥7,000,000 × 3 - ¥8,500,000 = ¥12,500,000
```

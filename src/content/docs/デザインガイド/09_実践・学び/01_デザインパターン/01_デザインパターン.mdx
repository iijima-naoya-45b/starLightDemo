## デザインパターン

デザインパターンは、よく使われるUIパターンの集合です。

### よく使われるデザインパターン

#### 1. カードパターン

```html
<div class="card">
  <img src="image.jpg" alt="Card image">
  <div class="card-content">
    <h3>Card Title</h3>
    <p>Card description</p>
    <button>Action</button>
  </div>
</div>
```

#### 2. モーダルパターン

```html
<div class="modal">
  <div class="modal-content">
    <h2>Modal Title</h2>
    <p>Modal content</p>
    <button>Close</button>
  </div>
</div>
```

#### 3. ナビゲーションパターン

```html
<nav>
  <ul>
    <li><a href="/">Home</a></li>
    <li><a href="/about">About</a></li>
    <li><a href="/contact">Contact</a></li>
  </ul>
</nav>
```

### まとめ

デザインパターンは、一貫性のあるUIを構築するための重要な要素です。

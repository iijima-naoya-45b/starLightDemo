## 特定商取引法詳細

### クーリング・オフ

#### クーリング・オフとは

クーリング・オフは、消費者が一定期間内であれば、理由なく契約を解除できる制度です。

#### クーリング・オフが適用される取引

- **訪問販売**: 事業者の営業所等以外の場所で行われる販売
- **通信販売**: 郵便、電話、インターネット等による販売
- **電話勧誘販売**: 電話による勧誘による販売

#### クーリング・オフの期間

- **訪問販売**: 契約日から8日間
- **通信販売**: 商品受領日から8日間（サービス提供開始日から8日間）
- **電話勧誘販売**: 契約日から8日間

#### 実践例: クーリング・オフの表示

```html
<div class="cooling-off">
  <h2>クーリング・オフについて</h2>
  <p>商品到着後8日以内であれば、理由なく契約を解除できます。</p>
  <p>クーリング・オフを希望される場合は、以下の方法でご連絡ください。</p>
  <ul>
    <li>電話: 03-1234-5678</li>
    <li>メール: support@example.com</li>
  </ul>
</div>
```

### 電子メール広告

#### 電子メール広告の規制

特定商取引法では、電子メール広告に関する規制があります。

#### 必要な表示事項

- **送信者情報**: 送信者の氏名または名称
- **連絡先**: 連絡先のメールアドレスまたは電話番号
- **広告である旨**: 「広告」などの表示

#### 実践例: 電子メール広告

```html
<!-- 電子メール広告の例 -->
<div class="email-ad">
  <p>【広告】</p>

  <p>連絡先: support@example.com / 03-1234-5678</p>
  <p>本文...</p>
</div>
```

## コンプライアンスチェックリスト

Webサイトを公開する前に確認すべきコンプライアンス項目のチェックリストです。

### 特定商取引法

- [ ] 事業者名が表示されている
- [ ] 所在地が表示されている
- [ ] 電話番号が表示されている
- [ ] 商品・サービスの価格が明示されている
- [ ] 送料が明示されている
- [ ] 支払方法が明示されている
- [ ] 返品・交換条件が明示されている
- [ ] クーリング・オフの説明がある

### 個人情報保護法

- [ ] プライバシーポリシーが設置されている
- [ ] 個人情報の利用目的が明示されている
- [ ] 個人情報の取得に関する同意がある
- [ ] 個人情報の安全管理措置が講じられている
- [ ] 個人情報の開示・訂正・削除の手続きが明示されている
- [ ] 第三者提供に関する説明がある

### 著作権法

- [ ] 使用している画像・コンテンツの利用許諾を確認している
- [ ] 出典が明示されている
- [ ] 著作権者の許可を得ている（必要な場合）

### 電子契約法

- [ ] 契約内容が明確に示されている
- [ ] 利用者の同意を確認している
- [ ] 契約の記録を保存している

### 景品表示法

- [ ] 誇大広告がない
- [ ] 根拠のない表現がない
- [ ] 比較対象が明確である
- [ ] 適切な注意書きがある

### AI関連法

- [ ] AI生成コンテンツの出典を明示している
- [ ] AIサービスの利用規約を確認し、遵守している
- [ ] AI利用に関する同意を取得している
- [ ] AI利用の目的を明示している
- [ ] 個人情報の安全管理措置を講じている（匿名化、暗号化）
- [ ] AI判断の根拠を説明できる
- [ ] バイアスや差別がないか確認している
- [ ] AI判断の監査ログを記録している

### まとめ

コンプライアンスチェックリストを確認することで、Webサイトの法的リスクを最小限に抑えることができます。

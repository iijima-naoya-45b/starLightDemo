---
title: "Expressフレームワーク"
label: "Expressフレームワーク"
---

Expressを使用して、Webアプリケーションを構築します。

```javascript
const express = require('express');
const app = express();

app.get('/', (req, res) => {
  res.send('Hello, Express!');
});

app.listen(3000, () => {
  console.log('Express server running on port 3000');
});
```

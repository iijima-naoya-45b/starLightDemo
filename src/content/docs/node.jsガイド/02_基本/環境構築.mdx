Node.jsの環境構築手順を以下に示します。

## インストール

Node.jsをインストールするには、nvmを使用することをお勧めします。以下のコマンドでnvmをインストールします。

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
```

その後、Node.jsをインストールします。

```bash
nvm install node
```

## セットアップ

Node.jsのバージョンを確認します。

```bash
node -v
```

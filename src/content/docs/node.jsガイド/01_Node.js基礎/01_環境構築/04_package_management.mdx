# パッケージ管理基礎

npmを使用した基本的なパッケージ管理を解説します。

## 1. 基本的なパッケージ管理

### パッケージのインストール

```bash
# パッケージのインストール
npm install express

# 開発依存としてインストール
npm install --save-dev typescript

# グローバルインストール
npm install -g nodemon
```

### package.jsonの作成

```bash
# package.jsonの作成
npm init -y
```

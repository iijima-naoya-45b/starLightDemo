# Express.js完全ガイド

Express.jsの実践的な実装方法を、実務で使える実装例とベストプラクティスとともに詳しく解説します。

## 1. Express.jsとは

### Express.jsの特徴

Express.jsは、Node.jsで最も広く使われているWebフレームワークです。

```
Express.jsの特徴
   ├─ シンプル
   ├─ 柔軟性
   ├─ ミドルウェア
   └─ ルーティング
```

## 2. 環境構築

### プロジェクトの作成

```bash
# プロジェクトの初期化
npm init -y

# Express.jsのインストール
npm install express

# TypeScriptの設定（オプション）
npm install -D typescript @types/express
```

## 3. 基本的なサーバー

```typescript
// server.ts
import express from 'express';

const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Hello, Express!');
});

app.listen(port, () => {
  console.log(`Server running on port ${port}`);
});
```

## 4. ルーティング

```typescript
// ルーティング
app.get('/users', (req, res) => {
  res.json({ users: [] });
});

app.post('/users', (req, res) => {
  const user = req.body;
  res.json({ user });
});

app.get('/users/:id', (req, res) => {
  const id = req.params.id;
  res.json({ id });
});
```

## 5. ミドルウェア

```typescript
// ミドルウェア
app.use(express.json());
app.use(express.urlencoded({ extended: true }));

// カスタムミドルウェア
app.use((req, res, next) => {
  console.log(`${req.method} ${req.path}`);
  next();
});
```

## まとめ

Express.js完全ガイドのポイント：

- **ルーティング**: REST APIの実装
- **ミドルウェア**: リクエストの処理
- **エラーハンドリング**: エラーの処理
- **テンプレート**: テンプレートエンジン

適切なExpress.jsの使用により、効率的なWebアプリケーション開発が可能になります。

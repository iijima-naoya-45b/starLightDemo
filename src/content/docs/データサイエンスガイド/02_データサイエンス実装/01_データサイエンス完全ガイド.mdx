## 2. データ分析

### データの可視化

```python
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# データの読み込み
df = pd.read_csv('data.csv')

# データの可視化
sns.scatterplot(data=df, x='x', y='y')
plt.show()
```

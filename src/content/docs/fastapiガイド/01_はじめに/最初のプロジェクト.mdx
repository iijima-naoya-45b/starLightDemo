---
title: "FastAPIの最初のプロジェクト"
label: "FastAPIの最初のプロジェクト"
---

FastAPIで最初のプロジェクトを作成するには、以下の手順を実行します。

1. `main.py`ファイルを作成します。

```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
async def root():
    return {"message": "Hello World"}
```

2. Uvicornを使用してサーバーを起動します。

```bash
uvicorn main:app --reload
```

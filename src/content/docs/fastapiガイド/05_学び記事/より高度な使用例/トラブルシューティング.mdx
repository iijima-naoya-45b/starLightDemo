FastAPIアプリケーションのトラブルシューティング方法を以下に示します。

## よくあるエラー

- **`HTTPException`**: リクエストが不正な場合に発生します。
- **`ValidationError`**: リクエストデータがバリデーションに失敗した場合に発生します。

## デバッグ

FastAPIでは、`print`関数を使用して、変数の値をデバッグできます。

```python
value = "Hello, World!"
print(value)
```

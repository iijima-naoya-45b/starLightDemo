FastAPIアプリケーションのパフォーマンス最適化方法を以下に示します。

## 非同期処理

FastAPIは、非同期処理をサポートしており、リクエストの処理を効率化します。

```python
from fastapi import FastAPI

app = FastAPI()

@app.get("/items/{item_id}")
async def read_item(item_id: int, q: str = None):
    return {"item_id": item_id, "q": q}
```

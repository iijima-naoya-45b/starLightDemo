FastAPIの環境構築手順を以下に示します。

## インストール

FastAPIをインストールするには、以下のコマンドを実行します。

```bash
pip install fastapi
```

## セットアップ

FastAPIアプリケーションをセットアップするには、以下の手順を実行します。

1. プロジェクトディレクトリを作成します。

```bash
mkdir my_fastapi_app
cd my_fastapi_app
```

2. 必要なパッケージをインストールします。

```bash
pip install fastapi uvicorn
```

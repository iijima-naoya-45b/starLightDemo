# Elixirの環境構築

Elixirをインストールするには、以下の手順を実行します。

## インストール

```bash
brew install elixir
```

このコマンドを実行することで、Elixirがインストールされます。

# 基本構文

Elixirの基本的な構文を紹介します。

## 変数と関数

```elixir
name = "Elixir"
IO.puts("Hello, #{name}!")

defmodule Math do
  def add(a, b) do
    a + b
  end
end

IO.puts(Math.add(1, 2))
```

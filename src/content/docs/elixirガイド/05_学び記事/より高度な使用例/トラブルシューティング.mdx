Elixirアプリケーションのトラブルシューティング方法を以下に示します。

## よくあるエラー

- **`FunctionClauseError`**: 関数の引数が一致しない場合に発生します。
- **`MatchError`**: パターンマッチングが失敗した場合に発生します。

## デバッグ

Elixirでは、`IO.inspect/2`を使用して、変数の値をデバッグできます。

```elixir
value = 42
IO.inspect(value, label: "Value")
```

# Railsの環境構築

Railsをインストールするには、以下の手順を実行します。

## インストール

```bash
gem install rails
```

このコマンドを実行することで、Railsがインストールされます。

# 基本構文

Railsの基本的な構文を紹介します。

## コントローラーの作成

```ruby
class WelcomeController < ApplicationController
  def index
    render plain: "Hello, Rails!"
  end
end
```

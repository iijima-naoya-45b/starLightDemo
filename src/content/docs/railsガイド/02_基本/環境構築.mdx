Railsの環境構築手順を以下に示します。

## Rubyのインストール

Rubyをインストールするには、rbenvを使用することをお勧めします。以下のコマンドでrbenvをインストールします。

```bash
brew install rbenv
rbenv init
```

その後、Rubyをインストールします。

```bash
rbenv install 3.0.0
rbenv global 3.0.0
```

## Railsのインストール

Railsをインストールします。

```bash
gem install rails
```

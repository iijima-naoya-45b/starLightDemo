## 環境構築

Railsでの開発を始めるには、まず開発環境を整える必要があります。ここでは、RubyとRailsのインストール手順を解説します。

### Rubyのインストール

Rubyのバージョン管理には「rbenv」の利用をおすすめします。これにより、複数のRubyバージョンを簡単に切り替えることができ、プロジェクトごとに最適なバージョンを使えます。

#### rbenvのインストール

ターミナルを開き、以下のコマンドを実行してrbenvをインストールします。macOSをご利用の場合はHomebrewが便利です。

```bash
brew install rbenv
rbenv init
```

`rbenv init`を実行すると、ターミナル起動時にrbenvが自動的に読み込まれるよう設定が行われます。指示に従って、`.zshrc`や`.bash_profile`などのシェル設定ファイルに追記してください。

#### Rubyのインストール

次に、rbenvを使ってRubyをインストールします。ここでは例としてバージョン3.3.0をインストールしますが、Railsガイドなどで推奨されている最新の安定版を選ぶと良いでしょう。

```bash
rbenv install 3.3.0 # 最新の安定版を指定
rbenv global 3.3.0 # このバージョンをシステム全体で使うように設定
```

### Railsのインストール

Rubyがインストールできたら、いよいよRailsをインストールします。

#### Railsのインストール

以下のコマンドを実行して、RailsをGemとしてインストールします。

```bash
gem install rails
```

これにより、Railsフレームワークとその依存関係がすべてインストールされます。

#### インストールの確認

インストールが完了したら、以下のコマンドでRailsのバージョンを確認し、正しくインストールされたかを確認しましょう。

```bash
rails -v
```

バージョン情報が表示されれば、Railsの開発環境構築は完了です！ 🎉 これで、Webアプリケーション開発の準備が整いました。

## 2. API統合

### REST API統合

```typescript
// レガシーシステムとの統合
class LegacySystemAdapter {
  async getData(id: string): Promise<any> {
    const response = await fetch(`http://legacy-system/api/data/${id}`);
    return response.json();
  }
  
  async sendData(data: any): Promise<void> {
    await fetch('http://legacy-system/api/data', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(data)
    });
  }
}
```

## まとめ

レガシーシステム統合完全ガイドのポイント：

- **API統合**: REST APIによる統合

- **アダプターパターン**: アダプターによる統合

適切な統合により、効率的なシステムを構築できます。

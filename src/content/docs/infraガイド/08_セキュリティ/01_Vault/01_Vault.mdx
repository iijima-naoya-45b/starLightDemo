## Vault

Vaultは、シークレット管理ツールです。APIキー、パスワード、証明書などの機密情報を安全に管理します。

### Vaultの基本概念

- **Secret**: 機密情報
- **Policy**: アクセス制御
- **Auth Method**: 認証方法

### 基本的な使用

```bash
# シークレットの保存
vault kv put secret/myapp/config username=admin password=secret

# シークレットの取得
vault kv get secret/myapp/config
```
